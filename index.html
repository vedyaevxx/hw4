Типи селекторів

h1{}
Працює по тегу. Будуть вибрані всі теги h1, які присутні на сторінці.
.class{}
Працює по класу. Будуть вибрані всі елементи, які мають однакове значення атрибутa "class".
#id{}
Працює по індентифікатору. Буде вибраний елемент з унікальним значенням ідентифікатора "id", яку може мати лише він.
*{}
Працює як універсальний. Значення, які будуть задаватися такому селектору - будуть примінятися до всіх елементів на сторінці.
[targed]{}
Працює по атрибуту. Bідбирають елементи по наявності атрибута або його значенню.
